1: Ensure that Windows is configured to enable the execution of scripts, if it is disabled follow the next steps:
	-> Initialize Windows Power Shell as administrator
	-> Run the next command: Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

2: GIT HUB

3: Move to the folder where you have downloaded the project
	->cd folder1\folder2\...\

4: Outside the streamlit_dashboard folder create an environment
	-> python3 -m venv .venv

5: Activate the environment
	-> . .\.venv\Scripts\Activate.ps1

6: Install the requirements
	-> pip install -r requirements.txt

7: Move inside the folder (here we will find the requirements.txt file)
	-> cd streamlit_dashboard

8: If you want to verify the requirements that you have installed you can extract the libraries versions that you have:
	-> pip freeze > requirements.txt

9: To run the Streamlit dashboard (inside the streamlit_dashboard folder) run the next code:
	-> streamlit run main.py

10: To quit the Stremlit dashboard:
	-> Ctrl + C

11: If you want to run only the jupyter notebook and you don't see the environment:
	->python -m ipykernel install --user --name=.venv --display-name ".venv"
